                                                                                          Redis Insight is a powerful desktop user interface that helps you visualize and optimize your data for Redis and Redis Stack. Plus it supports CLI interaction in a fully featured desktop UI client.

                                                                                          Build from anywhere

                                                                                          Redis Insight is supported on multiple operating systems: Linux, Windows, and macOS. It seamlessly supports all our deployments, whether you use Redis Open Source,Redis Stack,Redis Enterprise Software,Redis Cloud,Amazon ElastiCache, orAzure Cache for Redis.

                                                                                          > Enjoy the view

                                                                                          The Redis Insight graphic UI helps you browse and interact with your data.

                                                                                          • Filter and visualize Redis keys, perform CRUD operations, or delete keys in bulk.
                                                                                          • Use friendly keyboard navigation and display data in pretty-print JSON, hexadecimal, MessagePack, and many other formats.
                                                                                          • Use the Tree view to group data and enhance the navigation.

                                                                                          / Use intuitive CLI

                                                                                          Workbench, our advanced CLI, lets you run Redis commands with the built-in Monaco Editor. It has syntax highlighting, developer shortcuts, and auto-completion.
